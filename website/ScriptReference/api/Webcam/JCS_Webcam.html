JCS_Webcam

Simple and basic webcam class usage.


Variables


mDelay How fast the splash screen fade in?
mResumeTime How fast the splash screen fade out?

Functions


ActiveWebcam Start the webcam if the device detected on your machine.
TakeSnapShotWebcam Take a snap shot and save it into application data path.
Play Start the webcam action.
Pause Pause the webcam action.
Stop Stop the webcam action.
isPlaying Is the webcam currently playing?
