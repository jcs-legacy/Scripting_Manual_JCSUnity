# JCS_TriggerSwitchBGMEvent

Switch BGM when on trigger.


## Variables

<table>
  <tr>
    <td>mSoundClip</td>
    <td>Sound you want to load.</td>
  </tr>
  <tr>
    <td>mSoundFadeInTime</td>
    <td>Time to fade in the sound.</td>
  </tr>
  <tr>
    <td>mSoundFadeOutTime</td>
    <td>Time to fade out the sound.</td>
  </tr>
</table>
