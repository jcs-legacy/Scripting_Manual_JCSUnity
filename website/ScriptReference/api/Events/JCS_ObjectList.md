# JCS_ObjectList

List of gameobject holder, with some utility functions provided messing with list 
of gameobject.

## Variables

| Name     | Description      |
|:---------|:-----------------|
| mObjects | List of objects. |

## Functions

| Name                    | Description                            |
|:------------------------|:---------------------------------------|
| GetRandomObjectFromList | Returns a random object from the list. |
