# JCS_3DLiquidBar

3D liquid bar, do thing like health bar, mana ber, etc.


## Variables

<table>
  <tr>
    <td>mSpriteMask</td>
    <td>Sprite mask that mask out the inner bar sprite.</td>
  </tr>
  <tr>
    <td>mBarSpriteRenderer</td>
    <td>Please put the under texture bar here.</td>
  </tr>
</table>

  <tr>

  <tr>
