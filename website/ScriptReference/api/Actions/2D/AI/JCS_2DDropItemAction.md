  <h1>JCS_2DDropItemAction</h1>
</div>

<p>
  Action makes the gameobject drops item.
</p>


<br/>
<h2>Variables</h2>
<br/>

<table>
  <tr>
    <td>mDropWhenDies</td>
    <td>Drop once when the object is dead.</td>
  </tr>
  <tr>
    <td>mDropByTime</td>
    <td>Drop by time.</td>
  </tr>
  <tr>
    <td>mTimePerDrop</td>
    <td>Time to drop one time.</td>
  </tr>
  <tr>
    <td>mRandomTimeRange</td>
    <td>Effect the time every time it drops.</td>
  </tr>
</table>
