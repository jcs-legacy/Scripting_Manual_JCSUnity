# JCS_FreezeScaleAction

Action that freeze the gameobject scale.


## Variables

<table>
  <tr>
    <td>mActive</td>
    <td>Is this action active?</td>
  </tr>
  <tr>
    <td>mScaleToFreeze</td>
    <td>Scale where this gameobject freeze at.</td>
  </tr>
  <tr>
    <td>mFreezeScale</td>
    <td>Freeze the scale in each axis.</td>
  </tr>
</table>
