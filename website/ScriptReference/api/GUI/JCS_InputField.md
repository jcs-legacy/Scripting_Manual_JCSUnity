# JCS_InputField

A better version of input field handle for uGUI.


## Variables

<table>
  <tr>
    <td>mMaxLetters</td>
    <td>Max letters shown in the options.</td>
  </tr>
  <tr>
    <td>mDotCount</td>
    <td>How many dot shown to replace the letter.</td>
  </tr>
  <tr>
    <td>mApproachSec</td>
    <td>Dot is more approach to the second half of the text value.</td>
  </tr>
</table>
