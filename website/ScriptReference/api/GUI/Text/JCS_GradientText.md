# JCS_GradientText

Gradient text effect.

## Variables

<table>
<tr>
<td>mGradientType</td>
<td>Type of the gradient this text going to use to render.</td>
</tr>

<tr>
<td>mStartColor</td>
<td>Beginning color to render the text.</td>
</tr>

<tr>
<td>mEndColor</td>
<td>End color to render the text.</td>
</tr>
</table>
