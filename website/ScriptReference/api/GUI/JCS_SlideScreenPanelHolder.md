# JCS_SlideScreenPanelHolder

Slide panel holder.

# Variables

| Name            | Description                                    |
|:----------------|:-----------------------------------------------|
| mSlideFrictionX | How fast the this slide panel slide in x axis. |
| mSlideFrictionY | How fast the this slide panel slide in y axis. |

# Functions

| Name     | Description             |
|:---------|:------------------------|
| AddForce | Add Force to the panel. |
