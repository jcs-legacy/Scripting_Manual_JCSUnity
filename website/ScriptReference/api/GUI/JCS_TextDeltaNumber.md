<div id="content-header">
  <h1>JCS_TextDeltaNumber</h1>
</div>

<p>
  Like `JCS_DeltaNumber`, but instead of altering the sprite we
  alter text instead.
</p>

<br/>

  <tr>

  </tr>
  <tr>

  </tr>
  <tr>
    <td>mPostString</td>

  </tr>
  <tr>
    <td>mRoundPlace</td>
    <td>Place you want to round the decimal.</td>
  </tr>
  <tr>
    <td>mAnimNumberTime</td>
