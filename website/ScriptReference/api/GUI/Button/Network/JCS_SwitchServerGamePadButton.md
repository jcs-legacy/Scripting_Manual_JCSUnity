  <h1>JCS_SwitchServerGamePadButton</h1>
</div>

<p>
  Button provide the ability to change from one server to the other server.
</p>


<br/>
<h2>Variables</h2>
<br/>

<table>
  <tr>
    <td>mHost</td>

  </tr>
  <tr>
    <td>mPort</td>
    <td>Target Port to switch.</td>
  </tr>
  <tr>
    <td>mForce</td>
    <td>Force to switch server.</td>
  </tr>
</table>
