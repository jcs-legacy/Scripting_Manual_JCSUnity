<p>
  Record the velocity for the character controller component.
</p>

<br/>

    <td>MoveSpeed</td>
    <td>How fast it moves?</td>
  </tr>
  <tr>
    <td>SpeedFriction</td>

  </tr>
  <tr>

    <td>Freeze movement on x axis.</td>
  </tr>
  <tr>
    <td>FreezeY</td>
    <td>Freeze movement on y axis.</td>
  </tr>
  <tr>
    <td>FreezeZ</td>
    <td>Freeze movement on z axis.</td>
  </tr>
</table>
